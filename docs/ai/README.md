# Bynkbook AI Knowledge Pack

This directory contains the Bynkbook AI Knowledge Pack.

The pack is not model training. It is product knowledge and safety context that can later be wired into prompts, retrieval/RAG, admin-editable rules, or AI category suggestion context.

The pack defines:

- Core Bynkbook ledger truth and AI system rules.
- Category suggestion behavior and confidence expectations.
- Reconciliation definitions and review-first behavior.
- User-facing help style.
- Safety boundaries.
- Practical examples for common transactions.

Any future implementation must preserve Bynkbook product truth, sign rules, soft-delete exclusion, confirmation-first behavior, and safety boundaries.
